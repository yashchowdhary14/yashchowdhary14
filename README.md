### 🚀 Featured Projects

#### 🔍 [AI Resume-Job Matcher](https://github.com/yashchowdhary14/JD_RESUME_MATCHER)
*A smart matching engine utilizing NLP and vector databases to match candidates against target job descriptions.*
- **Vector Search**: Integrated FAISS index configurations (Flat, HNSW, IVF) for 384-dimensional sentence embeddings.
- **Skill Extraction**: Leveraged SentenceTransformers (`paraphrase-MiniLM-L12-v2`) to compute high-speed cosine similarities and extract keyword competencies.
- **Full Stack**: Designed clean Flask REST endpoints integrated with SQLAlchemy and a responsive candidate/recruiter dashboard.
- *Tech Stack*: `Python` `Flask` `FAISS` `SentenceTransformers` `SQLAlchemy` `Bootstrap`

#### 🎥 [Wav2Lip Video Resume Creator](https://github.com/yashchowdhary14/Video_Resume)
*A generative AI pipeline that creates professional talking head video resumes from audio files.*
- **Lip-Syncing Pipeline**: Integrated Wav2Lip deep learning models with the Sync SDK (Sync.so) API for high-resolution video sync.
- **Voice Synthesis Integration**: Syncs voiceovers with pre-recorded headshots automatically to generate natural video output.
- *Tech Stack*: `Python` `Wav2Lip` `Sync SDK` `PyTorch` `Deep Learning`

#### 💊 [Personalized Medical Recommendation System](https://github.com/yashchowdhary14/Medicine-Recommendation-System-Personalized-Medical-Recommendation-System-with-Machine-Learning-main)
*An ML-powered predictive healthcare assistant for symptom analysis.*
- **Disease Prediction**: Implemented Scikit-Learn classifiers (Random Forest, Decision Trees, etc.) trained to identify potential health conditions from input symptoms.
- **Recommendations**: Recommends top-rated medicines, precautions, and workout routines tailored dynamically to predicted conditions.
- *Tech Stack*: `Python` `Flask` `Scikit-Learn` `Machine Learning` `HTML5/CSS3`

#### 🤟 [Sign Language to Text & Speech Conversion](https://github.com/yashchowdhary14/Sign-Language-To-Text-and-Speech-Conversion)
*A real-time sign language translation tool that aids communication for the hearing and speech impaired.*
- **Landmark Detection**: Captures hand skeleton coordinates dynamically using OpenCV and MediaPipe.
- **Deep Learning Classifier**: Employs a custom CNN model in TensorFlow/Keras to achieve 97%-99% prediction accuracy across gestures.
- **Speech Synthesis**: Converts translated text output to real-time speech utilizing `pyttsx3`.
- *Tech Stack*: `Python` `OpenCV` `MediaPipe` `Keras/TensorFlow` `pyttsx3`

#### 🎙️ [Voice Assistant Jarvis](https://github.com/yashchowdhary14/Voice-Assistant-Jarvis)
*A smart virtual assistant built to automate day-to-day productivity tasks like checking calendars, auto-joining virtual meetings, and sending emails.*
- *Tech Stack*: `Python` `Tkinter` `PyAutoGUI`

#### 🚍 [Bus Booking Management System](https://github.com/yashchowdhary14/bus_booking_system)
*A convenient and efficient passenger ticket reservation and operator schedule management system.*
- *Tech Stack*: `JavaScript` `HTML` `CSS`
