### 🚀 Projects

#### 🔍 [JD Resume Matcher](https://github.com/yashchowdhary14/JD_RESUME_MATCHER)
An AI-powered recruiter intelligence system that matches resumes against job descriptions using vector search.
- Built with **Flask** and **FAISS** vector database supporting HNSW, Flat, and IVF indexing structures.
- Computes semantic cosine similarity over 384-dimensional embeddings generated by **SentenceTransformers**.
- Features automated skill-set extraction, build-time benchmarking, and a responsive recruiter portal.

#### 🎥 [Wav2Lip Video Resume Creator](https://github.com/yashchowdhary14/Video_Resume)
A generative AI video processing pipeline creating synchronized talking head video resumes from audio files.
- Synchronizes lip movements of headshot videos with vocal tracks using **Wav2Lip** deep learning models.
- Orchestrated media ingestion and synthesis endpoints integrated via the **Sync SDK** (Sync.so) API.

#### 🤟 [Sign Language to Text & Speech](https://github.com/yashchowdhary14/Sign-Language-To-Text-and-Speech-Conversion)
A real-time assistive communication engine translating hand gestures into spoken language.
- Tracks hand skeletons dynamically in real-time using **OpenCV** and **MediaPipe** landmarks.
- Employs a custom convolutional neural network (**CNN** in TensorFlow/Keras) achieving 97%–99% classification accuracy.
- Synthesizes real-time audio playback from text translations via **pyttsx3**.

#### 💊 [Personalized Medical Recommendation System](https://github.com/yashchowdhary14/Medicine-Recommendation-System-Personalized-Medical-Recommendation-System-with-Machine-Learning-main)
A predictive healthcare system recommending personalized therapeutics based on patient symptoms.
- Trained machine learning classifiers using **Scikit-Learn** for multi-symptom disease classification.
- Generates tailored drug descriptions, workout plans, and precautionary schedules dynamically via a **Flask** interface.
